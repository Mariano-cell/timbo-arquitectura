Map assets for Haras San Pablo map story.

Replace these files:
1) base-map.svg
   - Real base map (recognizable geography) covering the full initial view.
   - Preferred aspect ratio: 16:9 (same as the viewport).

2) region-overlay.svg
   - Transparent-background overlay with only the highlighted region/province shape.
   - Must be aligned to the same coordinate frame/aspect as base-map.svg.

When replacing, keep the same filenames to avoid changing HTML/JS paths.
